Telegram-бот прогнозирования тренда валют  
v2.0 — прогноз на основе реальных курсов ЦБ РФ  
 
+----------------------+--------------------------------------------------------+
|                      |                                                         |
| Цель проекта         | Прогноз направления движения курса (вверх/вниз) за день |

| Язык                 | Python 3.10+                                           |

| Библиотеки           | python-telegram-bot, requests                          |

| Источник данных      | ЦБ РФ (https://cbr.ru/scripts/XML_daily.asp)          |

+----------------------+--------------------------------------------------------+

Структура проекта

+-------------------------+----------------------------------------------------+
| v1_basic_bot/           | Версия 1.0: бот с прогнозом-заглушкой              |
| v2_data_fetch/          | Версия 2.0: бот с реальными данными ЦБ РФ          |
+-------------------------+----------------------------------------------------+

Запуск (v2.0)

1. Получите токен 
2. Установите зависимости:

   pip install python-telegram-bot==20.7 requests

3. В файле `v2_data_fetch/bot.py` замените:

   TOKEN = "ВАШ_ТОКЕН_ОТ_BOTFATHER"

4. Запустите:

   cd v2_data_fetch
   python bot.py

Логика прогноза (v2.0)

+----------------+-----------------------------------------------------------+
| Шаг            | Описание                                                  |
+----------------+-----------------------------------------------------------+
| 1. Получение   | Запрос курса валюты на вчерашний и позавчерашний день     |
|                | через API ЦБ РФ (XML)                                     |
+----------------+-----------------------------------------------------------+
| 2. Расчёт Δ    | Δ% = (курс_вчера – курс_позавчера) / курс_позавчера × 100 |
+----------------+-----------------------------------------------------------+
| 3. Прогноз     | → "вверх", если Δ% > 0                                    |
|                | → "вниз",  если Δ% < 0                                    |
|                | → "без изменений", если Δ% = 0                            |
+----------------+-----------------------------------------------------------+

Поддерживаемые валюты

+----------+-----------------+
| Валюта   | Код ЦБ РФ       |
+----------+-----------------+
| USD      | R01235          |
| EUR      | R01239          |
| CNY      | R01375          |
+----------+-----------------+

Команды:  
- `/predict USD`  
- `/predict EUR`  
- `/predict CNY`
